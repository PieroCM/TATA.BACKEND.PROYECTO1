## ¿Cómo Suscribirse a GitHub Copilot Pro?

Para suscribirte a GitHub Copilot Pro, sigue estos pasos:

### Opción 1: Desde GitHub.com

1. **Visita tu perfil de GitHub**
   - Ve a https://github.com/settings/copilot

2. **Selecciona un Plan**
   - GitHub Copilot Individual: $10 USD/mes o $100 USD/año
   - GitHub Copilot Pro: $10 USD/mes (mismo precio que Individual, pero con características adicionales)
   - GitHub Copilot Business: $19 USD/usuario/mes (requiere organización)

3. **Configura el Método de Pago**
   - Haz clic en "Start free trial" o "Subscribe"
   - Ingresa tu información de pago (tarjeta de crédito o débito)
   - Confirma tu suscripción

### Opción 2: Desde Visual Studio Code

1. **Instala la Extensión de GitHub Copilot**
   - Abre VS Code
   - Ve a la pestaña de Extensiones (Ctrl+Shift+X)
   - Busca "GitHub Copilot"
   - Haz clic en "Install"

2. **Inicia Sesión**
   - Cuando se te solicite, inicia sesión con tu cuenta de GitHub
   - Serás redirigido a GitHub para autorizar la extensión

3. **Suscríbete**
   - Si no tienes una suscripción activa, se te pedirá que te suscribas
   - Sigue las instrucciones para completar la suscripción

## Características de GitHub Copilot Pro

- Sugerencias de código más rápidas
- Acceso a modelos de IA más avanzados
- Mejor rendimiento en línea de comandos (Copilot CLI)
- Acceso prioritario a nuevas características

## Prueba Gratuita

GitHub ofrece una prueba gratuita de 30 días para nuevos usuarios. Durante este período, puedes probar todas las características sin costo.

## Soporte

Si tienes problemas con la suscripción:
- Visita: https://support.github.com
- O contacta a GitHub Support directamente desde tu cuenta
